A) ASP.Net Core Structure - top

This template is fully responsive and created with ASP.Net Core 8 and bootstrap4 framework. All the content is within wrapper. The wrapper is containing sidebar wrapper ,topbar header and content wrapper. In content wrapper all part is devided with row and cards. In this template most of the classes are used acording to bootstrap. Here is the general structure.

<!doctype html>
<html lang="en" data-bs-theme="blue-theme">
<head>
    @Html.Partial("~/Views/Shared/Title_Meta.cshtml")

    @RenderSection("styles", false)

    @Html.Partial("~/Views/Shared/Css.cshtml")

</head>
<body>
    @Html.Partial("~/Views/Shared/Topbar.cshtml")

    @Html.Partial("~/Views/Shared/Sidebar.cshtml")

    <!--start main wrapper-->
    <main class="main-wrapper">
        <div class="main-content">

            @Html.Partial("~/Views/Shared/Page_Title.cshtml")
            @RenderBody()
            @RenderSection("externalhtml", false)

        </div>
    </main>

    <!--end main wrapper-->
    <!--start overlay-->

    <div class="overlay btn-toggle"></div>

    <!--end overlay-->

    @Html.Partial("~/Views/Shared/Footer.cshtml")

    @Html.Partial("~/Views/Shared/Cart.cshtml")

    @Html.Partial("~/Views/Shared/Right_sidebar.cshtml")

    @Html.Partial("~/Views/Shared/Vendor_Scripts.cshtml")

    @RenderSection("scripts", required: false)

</body>

</html>

Maxton is build in Asp.Net Core web app using Visual Studio 2022. Please follow below steps to install and setup all prerequisites:
